  3. HTTP and HTTPS Protocols
As soon as you begin writing a URL, Uniform Resource Locator, the computer you are using begins communicating with another computer called a server that is usually thousands of miles away. The language they are communicating in is called HTTP, hypertext transfer protocol. The way it works is that the computer you are using to communicate to the server is sending a Get Request. For example, if you are trying to login through a login page, your computer sends a get request to the server and you receive all the HTML code in the page. HTML, hypertext markup language, is a language which is the building blocks of web pages.
